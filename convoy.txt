steps to install binary in linux:
$ echo "deb [trusted=yes] https://apt.packages.getconvoy.io/ /" | sudo tee -a /etc/apt/sources.list.d/convoy.list
$ sudo apt update
$ sudo apt install convoy




convoy has convoy cloud and convoy self hosting.
it supports mongodb as its datastore.
to run a mongo instance:
sudo service mongodb start
it supports redis as its queue.
sudo service redis-server start


to start convoy:
./convoy server --config ./convoy.json
